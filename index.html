Basics of the Web and HTML

Major Pieces of the Web

The web consists of 4 components: users, a computer with a browser, the internet, and servers.

The browser makes a request, via the internet, to servers. The internet communicates with the servers using HTTP. The servers respond with files which the browser displays.

See this video for a more in depth explanation of the web.

HTML

HTML is short for "HyperText Markup Language". HTML consists of text content, markup (tags and elements), references to other documents (images and video) and links.

A few examples of tags are bold, italics, anchor (for links) and image (this is a void tag- doesnt require a closing tag).

Tags can also have attributes. For instance, the anchor tag has the attribute href. This is used for creating links. Image tags have the attributes src (for url) and alt (for text when image doesnt load).

Inline vs. Block Elements

<br> (line break) is an inline element while <p> is a block element. Block elements form an "invisible box" around the content inside of them.

Span is inline while Div is a block element. These elements don't do anything besides contain content, but you can attach styles to them to affect the display of the content. You do this by adding CSS classes.
